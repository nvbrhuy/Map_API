# Get API google map execution steps:
1. Get API at https://console.cloud.google.com/
2. Clone this git file
3. Please add your API inside the "YOUR_API" script in the index.html file.
4. Example: 
5. <script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API&libraries=maps,marker" defer></script>
